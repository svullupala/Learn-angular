Getting Off to a Quick Start

To start using IBM Spectrum Protect Plus, you must complete steps that include defining the location of the resources that you want to protect and creating policies and jobs to run the backup operations for those resources. This getting started section provides the basic steps to set up and start using IBM Spectrum Protect Plus.

Before you start, ensure that you have installed the required components for IBM Spectrum Protect Plus including any additional vSnap servers and disks and VADP proxies.

For guidance about planning your storage environment, see the IBM Spectrum Protect Plus Blueprint.

For installation instructions, see Install IBM Spectrum Protect Plus as a VMware Virtual Appliance and Install vSnap Server.

In this section, the initial installation and configuration tasks are completed by the IBM Spectrum Protect Plus infrastructure administrator.

Then, hypervisor and application backup and restore tasks are completed by the application administrator as shown in the following table. However, a single administrator might be responsible for all tasks in your environment.

Action Owner Description
Start IBM Spectrum Protect Plus Infrastructure administrator and application administrator The IBM Spectrum Protect Plus administrator starts the application for the first time.
Create Backup Policies Infrastructure administrator Backup policies define the parameters that are applied to backup jobs. These parameters include the frequency and retention of backups and the option to offload backup data to secondary backup storage for longer-term protection. Backup policies are called service level agreement (SLA) policies in IBM Spectrum Protect Plus.
Create a User Account for the Application Administrator Infrastructure administrator User accounts determine the resources and functions that are available to the user. To create a user, you must log in using the default admin user ID, which is assigned to the SUPERUSER role, or a user ID that belongs to a role that provides the permission to create users.
Add Resources to Protect Application administrator Resources are servers for hypervisors or applications that host data that you want to protect.
Create a Backup Job Application administrator Backup jobs define the backup options for selected resource and the SLA policy or policies that are applied to the backup operation.
Start a Job Application administrator Jobs are started as defined in the SLA policy that is associated with the job. You can also manually start a job.

Run a Report

Application administrator IBM Spectrum Protect Plus provides a number of predefined reports that you can run with default parameters or modify to create custom reports.
